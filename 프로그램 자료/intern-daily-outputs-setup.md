# intern-daily-outputs 스케줄 태스크 설정 안내

이 파일은 **새 Cowork 대화창**에서 아래 메시지를 붙여넣으면 자동 설정됩니다.

---

## 새 대화창에 붙여넣을 메시지:

아래 내용으로 매일 평일 오후 5시에 실행되는 스케줄 태스크를 만들어줘:

- taskId: intern-daily-outputs
- 설명: 매일 오후 5시 - 인턴 3명(권미정/윤서연/박수빈) + 멘토(조준형) 개인별 맞춤 산출물 자동 생성
- cronExpression: 0 17 * * 1-5

태스크가 생성되면 이 파일(intern-daily-outputs-setup.md)을 삭제해도 됩니다.

---

설정 완료 후 생성되는 파일 목록 (매일 실습가이드 폴더에 저장):
- W{n}D{n}_권미정_실습가이드.docx  (인턴 개인 업무일지 포함)
- W{n}D{n}_윤서연_실습가이드.docx  (인턴 개인 업무일지 포함)
- W{n}D{n}_박수빈_실습가이드.docx  (인턴 개인 업무일지 포함)
- W{n}D{n}_조준형_멘토가이드.docx  (체크인표 + 피드백 양식)
