# 주차별 산출물 폴더 안내 (`deliverables_wN/`)

금요일 또는 멘토가 정한 날짜에 **한 폴더에 모아** 제출·공유할 때 사용합니다. (평일 **13:00–18:00** 인턴 업무시간·오프라인 일정은 계획서·멘토와 조정)

## 폴더 이름 예시

```
deliverables_w1/
├── README.md                 ← 이 주차 목록 (아래 표 복사)
├── journal/                  ← 일일 일지 (월~금)
│   ├── journal_2026-04-13.md
│   └── ...
├── goal_plan_w1d1.md
├── weekly_summary_w1.md
├── retrospective_w1.md
├── feedback_w1.md
└── (기타 산출물)
```

## 주차별 체크리스트 (복사해서 README에 붙여넣기)

| 항목 | 파일명 예시 | 포함 여부 |
|------|-------------|-----------|
| 일일 일지 5일분 | `journal/` | ☐ |
| 주간 요약 | `weekly_summary_wN.md` | ☐ |
| KPT 회고 | `retrospective_wN.md` | ☐ |
| 멘토 피드백 | `feedback_wN.md` | ☐ |
| 해당 주 핵심 과제 | (일정표의 산출물 이름) | ☐ |

---

## 파일 이름 규칙 (권장)

- 날짜: `YYYY-MM-DD`
- 주차: `w1` ~ `w8`
- 한글·공백은 가능하면 피하고, `_` 로 연결

---

## 백업

- 클라우드·GitHub **비공개** 저장소에 올릴 때는 `.env`·API 키·내부 URL이 포함되지 않았는지 확인합니다.
