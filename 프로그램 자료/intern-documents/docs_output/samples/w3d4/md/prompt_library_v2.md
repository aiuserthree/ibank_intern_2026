## 프롬프트 1: 감성 분류 (v2.0)

```
[PROMPT:sentiment_classify:v2.0]
당신은 텍스트 감성 분류 전문가입니다.
아래 텍스트를 POSITIVE / NEGATIVE / NEUTRAL 중 하나로 분류하고,
반드시 JSON 형식으로만 응답하세요.

### 예시 1
입력: "올해 매출이 목표를 20% 초과 달성했습니다"
출력: {"label": "POSITIVE", "confidence": 0.95, "signals": ["매출 초과 달성"]}

### 예시 2
입력: "인력 구조조정으로 500명이 해고됐습니다"
출력: {"label": "NEGATIVE", "confidence": 0.92, "signals": ["해고", "구조조정"]}

분류할 텍스트: {text}
[/PROMPT]
```

## 프롬프트 2: 3줄 요약 (v2.0)

```
[PROMPT:summarize_3line:v2.0]
다음 텍스트를 핵심 내용 중심으로 3줄 이내로 요약하세요.
- 첫 줄: 핵심 사실 (숫자 포함 시 그대로 유지)
- 둘째 줄: 맥락 또는 원인
- 셋째 줄: 시사점 또는 다음 단계

출력 형식:
{"line1": "...", "line2": "...", "line3": "..."}

텍스트: {text}
[/PROMPT]
```

---

## 프롬프트 3: CoT 분류 (v2.0)

```
[PROMPT:cot_classify:v2.0]
텍스트를 분류하기 전에 단계별로 추론하세요.

1단계: 핵심 감정 신호 단어를 찾는다
2단계: 긍정/부정/중립 여부를 판단한다
3단계: 확신도를 0~1로 추정한다

최종 출력만 JSON으로:
{"label": "POSITIVE|NEGATIVE|NEUTRAL", "confidence": 0.0~1.0}

텍스트: {text}
[/PROMPT]
```
