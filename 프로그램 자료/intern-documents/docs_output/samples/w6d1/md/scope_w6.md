## 1. 기획서 대비 범위 변경 사항

### 이번 스프린트에서 제외 (Descoped)
- **실시간 자동 수집 스케줄러**: API 키 발급 지연으로 W7 이후로 이월
  - 이유: 멘토 합의 — 데이터 품질이 더 중요, 스케줄러는 추가 기능
- **PDF 보고서 자동 추출**: pdfplumber 의존성 복잡도 증가, RSS+API로 충분
  - 이유: 500건 목표는 API+RSS만으로 달성 가능

### 이번 스프린트에 반드시 포함 (In Scope)
- [x] 공공 텍스트 수집 스크립트 (`collect_raw.py`) — 동작 확인
- [x] 전처리 파이프라인 (`preprocess.py`) — 단위 테스트 통과
- [x] LLM 분류 함수 (`classify.py`) — 배치 10건 성공
- [x] 출력 JSON 스키마 — 기획서와 동일한 필드명
