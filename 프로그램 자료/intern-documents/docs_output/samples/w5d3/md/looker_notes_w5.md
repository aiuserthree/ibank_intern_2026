## 1. 대시보드 목적

**독자:** 멘토 + 팀장 (비개발자)
**목적:** 공공기관 AI 도입 현황을 한눈에 파악

**핵심 KPI (3개)**

| KPI | 기준값 | 현재값 |
|-----|--------|--------|
| 분석 문서 수 | 100건 이상 | 124건 |
| 분류 정확도 | ≥ 85% | 80% (개선 중) |
| 카테고리 커버리지 | 5개 분야 | 5개 분야 ✅ |

---

## 2. 차트 구성

### 차트 1: 카테고리별 문서 수 (막대 차트)
- **X축:** AI 응용 분야 (5개 카테고리)
- **Y축:** 문서 수
- **인사이트:** 민원·행정 자동화(33%)가 가장 많음

### 차트 2: 월별 AI 언급 추이 (시계열)
- **X축:** 월 (2024-01 ~ 2026-04)
- **Y축:** 문서 수
- **인사이트:** 2025년 하반기부터 급격히 증가, 2026년 1분기 최고치

### 차트 3: 기관 유형별 분포 (도넛 차트)
- **분류:** 중앙부처 / 지자체 / 공공기관
- **인사이트:** 중앙부처 45%, 지자체 32%, 공공기관 23%

---

## 3. Looker Studio 연결 방법

```
1. Google Sheets에 output/report.json 데이터 붙여넣기
2. Looker Studio → 데이터 소스 추가 → Google Sheets 선택
3. 날짜 필드: pub_date (날짜 형식: YYYYMMDD)
4. 측정항목: count(doc_id)
5. 차원: category, source_type, pub_month
```

## 4. 대시보드 스토리

> "국내 공공기관의 AI 도입은 2025년 하반기를 기점으로 급증했으며, 민원·행정 자동화 분야가 전체의 1/3을 차지한다."

**문제 → 데이터 → 인사이트 → 액션:**
1. 문제: 공공기관 AI 도입 현황이 체계적으로 정리되지 않음
2. 데이터: 공공 보도자료 + 뉴스 124건 분석
3. 인사이트: 민원 자동화 집중, 의료·교육 분야 성장 여지
4. 액션: 의료·교육 분야 사례 추가 수집 권장

## 5. 한계

- 데이터: 124건은 통계적 대표성 부족 (목표: 500건 이상)
- 시각화: Looker Studio 무료 플랜 — 실시간 업데이트 불가
- 해석: 언급 빈도 = 실제 도입 수준이 아닐 수 있음
