## 빠른 선택 가이드

### 상황별 1분 판단 트리

```
요청 유형이 무엇인가?
├── 코드 작성·디버깅 → Cursor Agent (파일 컨텍스트 있으면 최강)
├── 로컬 파일·폴더 작업 → Claude Cowork
├── 최신 정보·뉴스 검색 → Genspark 또는 ChatGPT (웹 검색 모드)

└── 빠른 초안·짧은 응답 → ChatGPT
```

## 공통 주의사항

1. **PII 금지**: 이름·이메일·주민번호 등 개인정보는 어떤 툴에도 직접 입력하지 않음
2. **내부 자료 주의**: 기밀 문서는 로컬(Cursor/Cowork)에서만 처리
3. **모델 날짜 기록**: 비교 실험 시 반드시 모델명과 날짜 기록
4. **결과 검증**: AI 출력물의 수치·사실은 반드시 원본과 대조
