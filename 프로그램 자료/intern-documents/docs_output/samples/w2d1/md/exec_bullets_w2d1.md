## Executive Summary

**오늘의 핵심 메시지 (1줄):**
2026년 공공·행정 AI는 **RAG 기반 문서 검색 + 에이전트 자동화**로 빠르게 이동하고 있으며,
보안 요건 때문에 온프레미스 또는 VPC 배포 수요가 늘고 있다.

---

## 핵심 포인트 5개

- **RAG 확산:** 공공기관 FAQ·법령 검색에 벡터 DB + LLM 조합이 표준 패턴으로 자리잡는 중
- **에이전트 자동화:** 단순 챗봇에서 '파일 읽기 → 분석 → 보고서 초안 생성'까지 이어지는 다단계 에이전트로 발전
- **멀티모달:** 스캔 서식·이미지 포함 보고서를 LLM이 직접 처리 가능 → 행정 문서 디지털화 가속
- **비용 최적화:** 소형 모델(Haiku·Flash 계열) + 프롬프트 캐싱으로 API 비용 80% 이상 절감 가능
- **보안·온프레미스:** 민감 정보 취급 기관은 VPC 배포 또는 온프레미스 솔루션 검토 필수
