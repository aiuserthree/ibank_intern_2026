# 미니 프로젝트 기획서 — W5D5

**프로젝트명:** 공공기관 AI 활용 현황 분석 (2024~2026)
**날짜:** 2026-05-15 (금)
**일정:** W5D5 — 미니 프로젝트 기획서 & 주간 정리
**작성자:** 인턴
**버전:** v1.0

---

## 1. 배경

국내 공공기관의 AI 도입이 빠르게 늘고 있으나 체계적인 현황 분석이 부족하다. 글로벌 벤치마크(싱가포르·영국·에스토니아)와 비교하면 한국의 AI 도입은 민원 자동화에 편중되어 있으며, 의사결정 지원·의료·교육 분야로의 확장이 필요한 시점이다.

---

## 2. 목표

1. 공공 보도자료·뉴스 텍스트를 AI로 분류하여 분야별 현황 파악
2. 2024~2026년 트렌드 변화 시각화
3. 정책 시사점 2,000자 보고서 작성

---

## 3. 범위 (MVP)

**포함:**
- 공공데이터포털 보도자료 (500건 목표)
- 뉴스 기사 (300건 목표)
- 5개 분야 분류 (민원·데이터분석·챗봇·의료·교육)
- Looker Studio 대시보드 1종

**제외:**
- 비공개 내부 문서
- 해외 공공기관 원문 분석 (벤치마크만)
- 실시간 자동 수집

---

## 4. 방법론

| 단계 | 내용 | 도구 |
|------|------|------|
| 수집 | 공공 API + RSS 크롤링 | Python, feedparser |
| 전처리 | 텍스트 정제, PII 제거 | pandas, re |
| 분류 | Few-shot LLM 분류 | gpt-4o-mini |
| 시각화 | 카테고리별 트렌드 | Looker Studio |
| 보고서 | Claude로 인사이트 정리 | Claude Cowork |

---

## 5. 주차별 일정

| 주차 | 마일스톤 | 완료 기준 |
|------|----------|-----------|
| W5 | 기획서 v1.0 + 데이터 수집 착수 | 이 문서 작성 완료 |
| W6 | 분류 파이프라인 v1 + 데이터 500건 처리 | accuracy ≥ 85% |
| W7 | 대시보드 완성 + 보고서 초안 v1 | 2,000자 이상 |
| W8 | 최종 발표 자료 + 멘토 피드백 반영 | 발표 완료 |

---

## 6. 성공 지표

| 유형 | 지표 | 기준 |
|------|------|------|
| 정량 | 분류 정확도 | ≥ 85% (골드셋 50건) |
| 정량 | 처리 문서 수 | ≥ 500건 |
| 정성 | 보고서 완성도 | 멘토 4/5점 이상 평가 |

---

## 7. 윤리 & 보안

- 개인정보보호법 준수: PII 필드 수집 즉시 삭제
- 공개 데이터만 사용: 공공누리 유형 확인 후 사용
- AI 분류 결과는 사람이 샘플 검수 후 사용
- API 키: .env 파일 (git 제외), 팀 공유 시 비밀번호 관리자 사용

---
