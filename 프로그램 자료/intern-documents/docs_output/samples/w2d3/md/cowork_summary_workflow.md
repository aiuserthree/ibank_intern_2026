## 2. 사용 파일 및 프롬프트

### Step 1: 문서 읽기 & 표 구조화

- **입력 파일:** `industry_ai_cases_w2.md` (어제 작성한 사례 정리)
- **프롬프트:**
```
이 문서에서 공공·행정 AI 사례만 추려서,
산업 / 주요 기능 / 정량적 효과 열로 표를 만들어줘.
없는 수치는 N/A로 표기해줘.
```
- **결과:** *(표 붙여넣기 또는 요약)*

### Step 2: 인사이트 추출

- **프롬프트:**
```
위 표를 보고, 국내 공공기관이 AI를 도입할 때
공통으로 부딪히는 장벽 3가지를 찾고,
각 장벽에 대한 해결 방향을 한 줄씩 써줘.
```
- **결과:**
  1. 장벽: 개인정보 처리 제약 → 해결: 비식별화 후 처리 또는 온프레미스 배포
  2. 장벽: 예산·인력 부족 → 해결: 소형 모델 + API 비용 최적화
  3. 장벽: 기관 내부 수용성 저항 → 해결: 소규모 파일럿 성공 사례 먼저 확보
