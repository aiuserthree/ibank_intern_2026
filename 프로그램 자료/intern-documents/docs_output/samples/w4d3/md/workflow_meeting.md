# 회의록 요약 워크플로우 — W4D3

**날짜:** 2026-05-06 (수)
**일정:** W4D3 — 오픈클로 템플릿·워크플로우 설계
**작성자:** 인턴

---

## 워크플로우 개요

**목적:** 회의 텍스트 노트를 구조화된 회의록으로 자동 변환

**트리거:** 수동 (회의 종료 직후)

**흐름:**
1. 회의 메모 텍스트를 `data/meeting_raw.txt`에 저장
2. Cowork에 요약 요청
3. 결과: 의제별 분류 + 결정사항 + 액션 아이템
4. `reports/meeting_YYYY-MM-DD.md`에 저장
5. 참석자에게 이메일 공유 (수동)

---

## Cowork 프롬프트

```
다음 회의 메모를 아래 형식으로 구조화하세요.

## 결정사항
- (결정된 내용만)

## 액션 아이템
| 담당자 | 내용 | 마감 |
|--------|------|------|

## 논의 내용 요약
- (의제별 핵심만)

회의 메모: {raw_notes}
```

---
