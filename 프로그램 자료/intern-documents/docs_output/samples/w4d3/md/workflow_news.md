# 뉴스 브리핑 워크플로우 — W4D3

**날짜:** 2026-05-06 (수)
**일정:** W4D3 — 오픈클로 템플릿·워크플로우 설계
**작성자:** 인턴

---

## 워크플로우 개요

**목적:** 매일 AI/정책 관련 뉴스를 자동으로 수집·요약하여 이메일로 수신

**트리거:** 매일 07:00 (스케줄)

**흐름:**
1. RSS 피드 수집 (`config/news_keywords.json` 참조)
2. 제목+본문 앞부분을 Cowork에 전달
3. Cowork: "2~3줄 한국어 요약, 핵심 수치 유지" 프롬프트
4. 결과를 `reports/news_YYYY-MM-DD.md` 저장
5. 이메일 발송 (수동 검토 후)

---

## 입력 스키마

```json
{
  "date": "2026-05-06",
  "keywords": ["AI", "LLM", "정책", "자동화"],
  "max_articles": 5,
  "summary_length": "2-3줄"
}
```

---

## Cowork 프롬프트

```
다음 뉴스 기사를 2~3줄로 요약하세요.
- 핵심 사실과 숫자는 그대로 유지
- 출처(매체명)를 첫 줄에 표기
- 한국어, 짧고 명확하게

기사: {article_text}
```

## 오류 처리

| 상황 | 처리 |
|------|------|
| RSS 피드 다운 | 이전 날 캐시 사용 + Slack 알림 |

| 키워드 매칭 0건 | 빈 리포트 + 경고 이메일 |
