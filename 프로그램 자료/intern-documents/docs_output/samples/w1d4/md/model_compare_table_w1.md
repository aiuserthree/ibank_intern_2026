## 1. 주요 모델 비교

| 구분 | **GPT-4o / o 시리즈 (OpenAI)** | **Claude 3.x / 4.x (Anthropic)** | **Gemini 2.x (Google)** |
|------|-------------------------------|-----------------------------------|-------------------------|
| **한 줄** | 범용 멀티모달·광범위 에코시스템 | 긴 맥락·문서 분석·추론·코드 읽기 강점 | 검색·워크스페이스·구글 생태계 연계 |
| **컨텍스트** | 128k ~ 200k | 200k+ | 1M+ (일부 모델) |
| **에이전트** | GPTs · Assistants API · Function Calling | Tool Use · Cowork · Claude Code | Gemini API · Vertex AI Agent |
| **추론 강화** | o1 · o3 시리즈 (extended thinking) | Extended Thinking (Claude 3.7+) | Gemini Thinking 모드 |
| **강점** | API 생태계 넓음, 응답 속도, 제품군 다양 | 장문 맥락, 지시 준수·분석·코딩 | 검색 결합·멀티모달·긴 컨텍스트 |
| **주의할 점** | 데이터 처리 정책·비용·할당량 | 가용 리전·API 비용 | 기업 데이터 연계 시 권한·보관 정책 |
| **공공·행정 적합 예시** | 민원 챗봇·문서 분류 | 긴 정책 문서 요약·계약 분석 | 검색+문서 혼합 Q&A |

## 3. 공공·행정·디지털 정부 맥락 키워드
