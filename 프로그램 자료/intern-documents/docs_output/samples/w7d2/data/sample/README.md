## 주의사항

- 모든 데이터는 **합성(synthetic)** 데이터입니다.
- 실제 기관명·담당자명은 가상으로 치환됐습니다.
- 원본 수집 데이터(data/raw/)는 이 저장소에 포함되지 않습니다.
- 재배포 시 라이선스(LICENSE) 파일을 확인하세요.
