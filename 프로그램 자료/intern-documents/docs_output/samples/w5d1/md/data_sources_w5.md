## 1. 주요 데이터 소스

### 소스 1: 공공데이터포털 보도자료
- **URL:** https://www.data.go.kr
- **유형:** 텍스트 (HTML/JSON API)
- **기간:** 2024-01-01 ~ 2026-04-30
- **예상 건수:** 약 500건 (키워드: AI, 인공지능, 디지털 전환)
- **저작권:** 공공누리 제1유형 (출처 표시 후 자유 이용)
- **수집 방법:** 공공데이터포털 Open API

### 소스 2: 행정안전부 디지털정부 보고서
- **URL:** https://www.mois.go.kr
- **유형:** PDF
- **기간:** 2024~2025 연간 보고서
- **예상 건수:** 약 20건
- **저작권:** 공공저작물 (출처 명시)
- **수집 방법:** 수동 다운로드 + pdfplumber 추출

### 소스 3: 뉴스 기사
- **출처:** 뉴시스, 연합뉴스 공개 RSS
- **유형:** 텍스트 (RSS/HTML)
- **기간:** 2024-01-01 ~ 2026-04-30
- **예상 건수:** 약 300건
- **저작권:** 뉴스 저작권법 준수 — 요약·분석 목적 인용
- **수집 방법:** feedparser + BeautifulSoup
