## Problem — 개선이 필요한 점

**1. 수집 건수 목표 미달**
500건 목표 중 387건에 그쳤다. 공공 API 속도 제한과 RSS 피드 중복 건 필터링에 예상보다 시간이 걸렸다.

**2. 분류 정확도 82%로 목표 미달**
85% 목표를 달성하지 못했다. 카테고리 경계 케이스(민원↔챗봇)에서 오류가 집중됐다.

**3. 시각화가 마지막 날로 밀림**
W6D5에 PPT 초안과 회고를 동시에 하다 보니 차트 구현이 충분하지 않았다.
