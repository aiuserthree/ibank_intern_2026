# 미니 프로젝트 기획서 (확정 — VOC 대시보드)

- **주제:** **서비스 리뷰·VOC AI 분석 대시보드** (계획서 §4 후보 ①)  
- **수행:** 아이뱅크 인턴 **개인 미니 프로젝트** (기획·데이터·파이프라인·시각화를 본인이 주도)  
- **멘토:** 조준형  
- **기간(가정):** 5~8주차 집중 구현·검증  

---

## 1. 배경·문제

앱·웹 서비스 **리뷰·VOC**가 채널별로 흩어져 있어, 톤·이슈 유형을 빠르게 파악하기 어렵다. 공개 리뷰(이용약관 준수)만으로 **감성·VOC 토픽**을 자동 분류하고 **대시보드**에서 추이·집중 이슈를 보고자 한다.

---

## 2. 목표

| 정량 | 정성 |
|------|------|
| 리뷰 N건 처리·JSON 스키마 고정 | 재현 가능한 파이프라인·README |
| 토픽·감성 일치율 목표치(멘토·본인 합의) | Looker/웹에서 동일 지표 재현 |

---

## 3. 범위

- **포함:** 스토어 리뷰 수집(합법)·전처리·감성·VOC 토픽 분류·`report.json`·주간 브리핑 MD·대시보드(Looker 또는 HTML/Next.js).  
- **제외:** 비공개 민원 DB, 약관 위반 수집, 개인 식별 가능한 리뷰 무단 전재.
