# 데이터 출처 메모 — **VOC 대시보드**

**프로그램:** AI 업무 경험 · 아이뱅크 · **개인** 미니 프로젝트 주제 **서비스 리뷰·VOC AI 분석 대시보드**

| 단계 | 내용 |
|------|------|
| **샘플 CSV** | `data/sample_voc_reviews.csv` — 교육용 가공 리뷰(가상 앱명·평점·문장). 실제 스토어와 무관. |
| **실제 과제** | **Google Play / App Store** 공개 리뷰·공식 API 또는 수출 규정에 맞는 수집. **이용약관·저작권·개인정보 최소화**·수집 일자 기재. |
| **저장** | `data/raw.csv`는 `collect_raw.py`로 복사·검증 후 파이프라인 입력. |

## 필수 컬럼 (스키마)

`id`, `date`, `source_type`, `app_name`, `rating`, `review_text`
